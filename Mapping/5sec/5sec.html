Options there's probably more than you'll need

Map options, configurable per map in shortcodes or map builder;

  • width & height
  • map color & skin - 10 predefined skins + unlimited colors
  • zoom
  • map type - road, terrain, satellite, hybrid
  • info layers - traffic, transit, weather, clouds, bicycle
  • fullscreen
  • disable mouse scroll wheel
  • autofit - adjusts map zoom and center to fit all pins

Pin options, configurable per pin in shortcodes or map builder;

  • address - human-readable address and lat/lng coordinates
  • description - large bubble available on click
  • auto open description
  • tooltip - small description
  • center - use this pin as map's center
  • bounce - animates pin
  • directions - enable directions form
  • icon - 25 predefined pin icons + custom ones

Advanced options amongst other things include the ability to use any custom field as a value inside the map. So instead of writing the address on every page you can just do: [pin]$my-saved-address[/pin] and the value will be taken from the custom field.
Pin description can be taken from the content of any page or post. That way you can design it and enter a large amount of content then simply use it in multiple pins.
