jQuery Facebook Wall Plugin

Put your Facebook page’s wall or timeline on your website. You always wanted to… Now you can! jQueryFacebookWall is a jQuery plugin that lets you add a Facebook wall feed to any page on your website. It’s easy! The jQuery Facebook wall plugin provides you all the tools you’ll need to add a wall or timeline style Facebook feed in minutes.

VERSION 2.0 Released

Now you can put multiple feeds on the same page! Or split up the feed, photos, albums, likes and events components and place them in different parts of your page.

Version 2 Changes

  1. Multiple Plugin instances on the same page. This allows for multiple feeds from different Facebook pages on the same page of your website.
  2. Dividable the plugin components (photos, albumns, feed, likes, events). You can divide the plugin components into different plugin instances and place them in different parts of your webstie page
  3. Past events display options.
  4. Display Animation
  5. Facebook Graph API calls and Login updated to API version 2.1
  6. Facebook profiles and groups are no longer supported. With the upgrade to Graph API 2.1 permission to personal profiles data via apps has been severely restricted. Please see these links for info on the difference between Facebook pages and Facebook profiles and how to convert your profile to a page.
